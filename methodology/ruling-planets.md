## Rule RP-RSC1: Four-Fold Ruling Planets from Both Moon and Lagna

**Source:** R.S. Chani, Cuspal System BTR (Point 1)
**URL:** https://www.futuresamachar.com/hi/birth-time-rectification-cuspal-system-8042

**Rule:** At the time of analysis, compute TWO sets of 4 Ruling Planets each — one from the Moon's position (RP Moon) and one from the Lagna's position (RP Lagna). Each set comprises:
- **Sign Lord** (Rashi lord)
- **Star Lord** (Nakshatra lord)
- **Sub Lord**
- **Sub-Sub Lord**

**Cross-fixing principle:** RP Moon's 4 planets are used to fix the Lagna at birth. RP Lagna's 4 planets are used to fix the Moon at birth.

**Worked Example (13 July 2016, 10:48:26 AM, Delhi):**
- **RP Lagna** (at query): Mercury-Sun-Saturn-Moon → Lagna was rising in Mercury's sign, Sun's star, Saturn's sub, Moon's sub-sub
- **RP Moon** (at query): Venus-Rahu-Jupiter-Mars → Moon was transiting Venus' sign, Rahu's star, Jupiter's sub, Mars' sub-sub
- Rahu represented Sun and Venus because Rahu was in Leo (Sun's sign) and Venus' nakshatra on that day

**Application:** These 8 ruling planets (4 from Moon + 4 from Lagna) form the complete RP set for BTR. All subsequent adjustments must respect these rulers.

**Houses/Planets:** Lagna, Moon, all cuspal divisions (sign/star/sub/sub-sub)

## Rule RP-RSC2: Fix Moon First When Large Time Uncertainty

**Source:** R.S. Chani, Cuspal System BTR (Point 2)
**URL:** https://www.futuresamachar.com/hi/birth-time-rectification-cuspal-system-8042

**Rule:** When the birth time uncertainty is large (e.g., 30 minutes or more), FIRST fix the Moon's position using RP Lagna, THEN fix the Lagna using RP Moon. The Moon is the slower-moving factor and provides the initial constraint.

**Priority order:**
1. RP Lagna (4 planets) → Fix Moon's cuspal position (sign/star/sub/sub-sub)
2. RP Moon (4 planets) → Fix Lagna's cuspal position (sign/star/sub/sub-sub)

**Application:** Always begin with the Moon correction when the birth time window exceeds the sub-lord sensitivity range (~5-12 minutes).

## Rule RP-RSC3: Start at Midpoint of Given Time Range

**Source:** R.S. Chani, Cuspal System BTR (Point 3)
**URL:** https://www.futuresamachar.com/hi/birth-time-rectification-cuspal-system-8042

**Rule:** When given a time range (e.g., 6:00–6:30 AM), start the calculation at the MIDPOINT of the range. This gives equal room to move forward or backward as needed.

**Formula:** `Start_Time = (Earliest_Time + Latest_Time) / 2`

**Worked Example:** Range 6:00–6:30 AM → Start at 6:15 AM. From this point, the astrologer can search forward and backward freely without bias.

## Rule RP-RSC4: Check Initial Moon Position Against RP Lagna

**Source:** R.S. Chani, Cuspal System BTR (Points 4-6)
**URL:** https://www.futuresamachar.com/hi/birth-time-rectification-cuspal-system-8042

**Rule:** At the starting time (midpoint), compute the Moon's cuspal position (sign lord - star lord - sub lord - sub-sub lord). Check how many of the 4 RP Lagna planets appear in this position.

**Evaluation:**
- If all 4 RP Lagna planets are found up to the SUB level → chart appears correct to sub level
- If the STRONGEST RP Lagna planet (Moon, being Lagna lord at query) is NOT found as the Moon's Sub-Sub Lord → time must be adjusted

**Worked Example:**
- At 6:15 AM: Moon position = Mercury-Sun-Saturn-Rahu (Sign-Star-Sub-SubSub)
- RP Lagna = Mercury-Sun-Saturn-Moon
- Mercury, Sun, Saturn all present up to sub level ✓
- BUT Moon's Sub-Sub Lord = Rahu (not Moon, the strongest RP) ✗
- Therefore: Time MUST be adjusted

**Strongest RP Priority:** In RP Lagna, the Lagna lord at query (in this case, Moon) is the strongest ruling planet. The Moon's Sub-Sub Lord at birth MUST match this strongest RP.

## Rule RP-RSC5: Adjust Time Until Strongest RP Appears as Moon's Sub-Sub Lord

**Source:** R.S. Chani, Cuspal System BTR (Points 7-8)
**URL:** https://www.futuresamachar.com/hi/birth-time-rectification-cuspal-system-8042

**Rule:** Move the birth time backward (earlier) until the strongest RP Lagna planet (the Lagna lord at query time) appears as the Moon's Sub-Sub Lord. The time range where this condition holds defines the candidate birth window.

**Procedure:**

## Method: Decision Planets (KP BTR)
**Source:** Rectification of Birth Time (KP) by K Hariharan
**Rule:** The decision planets constantly differ with time and place of judgment. Unlike static birth chart analysis, the decision planets shift depending on when and where the rectification attempt is made. The astrologer must recompute the decision planets at the actual time and location of the correction effort.

**Key Principle:** The decision planets are not fixed to the birth moment alone — they reflect the current moment of judgment. This means a rectification done at different times or locations may yield different ruling-planet sets, and the astrologer must account for this variability.

**Application:** When commencing birth time rectification, note the exact time and place. Compute the decision planets for that moment. Use these — not the birth moment planets — as the reference set for matching.
